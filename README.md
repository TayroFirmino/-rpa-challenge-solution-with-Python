# rpa-challenge-solution-with-Python
Este é um projeto de demonstração para automação do desafio RPA Challenge usando Python e Playwright. O objetivo principal é inputar os dados necessários na página web.

## 📋 Pré-requisitos
Certifique-se de ter o Python instalado em seu sistema. Caso contrário, você pode baixá-lo [aqui](https://www.python.org/downloads/).

## 🔧 Instalação
Clone este repositório em sua máquina local:
```bash
git clone https://github.com/TayroFirmino/rpa-challenge-solution-with-Python.git
```
## 🚀 Começando
Execute o script start.sh
```bash
sh start.sh
```

Este script irá:

Verificar se o ambiente virtual (venv) já está criado. Se não, irá criá-lo.
Ativar o ambiente virtual.
Instalar as dependências necessárias listadas no arquivo requirements.txt.
Instalar o chromium para o Playwright
Executar o script principal main.py que irá automatizar o desafio RPA.
